# Dynamic Wind Physics Boat Simulation

This example demonstrates a simplified wind physics model where a boat's speed is dynamically influenced by its heading relative to the wind direction. Players can steer the boat, observing how wind changes and boat orientation strategically impact movement, mirroring the core concept of the TinyWind game.

## Language

`html`

## How to Run

Save the code as `index.html`. Open `index.html` in any modern web browser. Use the Left and Right arrow keys to steer the boat.
